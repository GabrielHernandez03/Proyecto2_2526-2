este proyecto consiste en programar un sistema en java que simula una cola de impresion usando un monticulo binario para las prioridades y una tabla de dispersion para buscar usuarios rapido . la idea es gestionar documentos segun el tiempo y el tipo de usuario, permitiendo ver todo de forma grafica y en arbol sin usar librerias para las estructuras
